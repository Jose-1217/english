# Student Assessment Program in Python

## Description

This Python program allows you to record a student's data, calculate their grade point average (GPA), and determine whether they pass or fail based on the result.

## Data Requested by the Program

The program asks the user to enter:

* Student's Name
* Grade (1st to 3rd)
* Three of the student's grades

## Functionality

1. The user enters the student's name and grade.

2. Then, they enter three grades.

3. The program calculates the average of the three grades.

4. It evaluates whether the student passes or fails.

5. Finally, a report with the student's data is displayed.

## Evaluation Rule

* If the average is **greater than or equal to 3.0**, the student **passes**.

* If the average is **less than 3.0**, the student **fails**.
